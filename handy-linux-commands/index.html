Turn a folder of PNGs into a video (requires mencoder):

  1. cd into the image directory
  2. mencoder mf://*.png -mf type=png:w=800:h=600:fps=1 -ovc lavc -lavcopts vcodec=mpeg4 -oac copy -o output.avi This command makes a video 800x600 pixels at one frame per second. You can omit the dimensions or change file names, and change the framerate. Taken from ubuntuforums.
