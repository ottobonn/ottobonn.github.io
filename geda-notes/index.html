gEDA Notes

In the course of using the *nix-based gEDA for my electronics designs, I have learned some ups and downs of the suite. I am recording my notes on this page in the hopes that they may help others using gEDA/gaf.

Notes on PCB

  • When using custom elements, must hand-edit xgsch2pcb project file to indicate directory of custom component library
  • Using a custom PCB footprint, updating the footprint does not update the layout. Must also edit the .pcb file to include the new element definition.
